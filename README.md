🚀 A high-performance, SEO-optimized personal portfolio and technical blog built with **Hugo**, featuring a modern glassmorphism UI and minimalist dark theme.

🌐 **Live Site:** https://snehbera.in

## ✨ Features

* ⚡ Ultra-fast static site generation using Hugo
* 🎨 Custom glassmorphism UI design
* 🌙 Clean and minimal dark theme
* 📱 Fully responsive across all devices
* 🔍 SEO optimized for better visibility
* 🔎 Image zoom support for blog content
* 📝 Integrated technical blogging system
* 🚀 CI/CD deployment with Vercel
* 🔐 Secure and scalable architecture

## 🛠️ Local Development Setup

### 1️⃣ Install Hugo Extended

Download from: https://gohugo.io/installation/

Verify installation:

```bash
hugo version
```

### 2️⃣ Clone the Repository

```bash
git clone https://github.com/Snehbera/snehbera.in.git
cd snehbera.in
```

### 3️⃣ Run Development Server

```bash
hugo server -D
```

👉 Open in browser: http://localhost:1313

### 4️⃣ Build for Production

```bash
hugo
```
